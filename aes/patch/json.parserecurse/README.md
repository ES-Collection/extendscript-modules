# json.parserecurse

> Extension

The `JSON.parseRecurse()` method is an alternate `JSON.parse()` function that uses recursive descent instead of eval.

Source: [JSON-js](https://github.com/douglascrockford/JSON-js/blob/master/json_parse.js)
