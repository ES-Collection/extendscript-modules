# object.keys

> ES5 Shim

The `Object.keys()` method returns an array of a given object's own enumerable properties.

Source: [ps-scripting-es5shim](https://github.com/EugenTepin/ps-scripting-es5shim/blob/master/lib/Object/keys.js)
